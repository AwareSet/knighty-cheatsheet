🌿 Git

Ultimate Interactive Cheat Sheet for Version Control
🚀 Setup & Config
Configure User Name
git config --global user.name "Your Name"
Set the name for your commits.
Configure User Email
git config --global user.email "your.email@example.com"
Set the email for your commits.
Initialize Repository
git init
Create a new local repository.
Clone Repository
git clone /path/to/repository
Create a working copy of a local repository.
Clone Remote Repository
git clone username@host:/path/to/repository
Create a working copy of a remote repository.
📦 Staging & Committing
Check Status
git status
List the status of your files.
Add File to Staging
git add <file>
Add one file to the staging area.
Add All Changes
git add .
Add all new and changed files to staging.
Commit Changes
git commit -m "Commit message"
Commit your staged changes with a message.
Commit and Add
git commit -a -m "Commit message"
Add all tracked files and commit.
🌿 Branching & Merging
List Branches
git branch
List all local branches.
Create New Branch
git branch <branch-name>
Create a new branch.
Switch to Branch
git checkout <branch-name>
Switch to another branch.
Create and Switch
git checkout -b <branch-name>
Create a new branch and switch to it.
Merge Branch
git merge <branch-name>
Merge a branch into the active branch.
Delete Branch
git branch -d <branch-name>
Delete a local branch.
☁️ Remote Repositories
List Remotes
git remote -v
List all configured remote repositories.
Add Remote
git remote add <name> <url>
Add a new remote repository.
Fetch from Remote
git fetch <remote-name>
Fetch changes from the remote, but don't merge.
Pull from Remote
git pull <remote-name> <branch-name>
Fetch and merge changes from the remote.
Push to Remote
git push <remote-name> <branch-name>
Push your changes to the remote.
📜 History & Undoing
View Log
git log
View the commit history.
View Log (Pretty)
git log --oneline --graph --decorate
A more compact and graphical log view.
Unstage File
git reset <file>
Remove a file from the staging area.
Revert File
git checkout -- <file>
Discard changes in your working directory.
Revert Commit
git revert <commit-hash>
Create a new commit that undoes a previous commit.
Reset to Commit
git reset --hard <commit-hash>
Reset your HEAD pointer to a previous commit (destructive).
✨ Advanced
Rebase
git rebase <branch-name>
Apply commits from current branch on top of another.
Interactive Rebase
git rebase -i HEAD~3
Interactively edit the last 3 commits.
Stash Changes
git stash
Temporarily save modified, tracked files.
Apply Stash
git stash pop
Apply the last stashed changes and remove it from the stash list.
Cherry Pick
git cherry-pick <commit-hash>
Apply a specific commit from another branch.
