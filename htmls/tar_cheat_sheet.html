Tar Cheat Sheet

Archive Creation, Extraction & Compression

üì¶ Basic Archive Operations

tar -cf archive.tar files/
Create archive
tar -tf archive.tar
List archive contents
tar -xf archive.tar
Extract archive
tar -rf archive.tar newfile.txt
Append files to archive
tar -uf archive.tar files/
Update archive with newer files

üóúÔ∏è Compression Options

tar -czf archive.tar.gz files/
Create gzip compressed archive
tar -cjf archive.tar.bz2 files/
Create bzip2 compressed archive
tar -cJf archive.tar.xz files/
Create xz compressed archive
tar -czf archive.tgz files/
Create gzip archive (.tgz format)
tar -caf archive.tar.gz files/
Auto-detect compression by extension

üì§ Extraction Options

tar -xzf archive.tar.gz
Extract gzip archive
tar -xjf archive.tar.bz2
Extract bzip2 archive
tar -xJf archive.tar.xz
Extract xz archive
tar -xf archive.tar -C /path/to/dest/
Extract to specific directory
tar -xf archive.tar file1 file2
Extract specific files only

üëÅÔ∏è Verbose & Progress

tar -cvf archive.tar files/
Create with verbose output
tar -xvf archive.tar
Extract with verbose output
tar -tvf archive.tar
List with detailed information
tar -czf archive.tar.gz files/ --checkpoint=1000
Show progress every 1000 files
tar -czf archive.tar.gz files/ --checkpoint-action=echo
Show progress with echo action

üö´ Exclude & Include

tar -czf archive.tar.gz --exclude="*.tmp" files/
Exclude files by pattern
tar -czf archive.tar.gz --exclude-from=exclude.txt files/
Exclude patterns from file
tar -czf archive.tar.gz --exclude="*.log" --exclude="*.tmp" files/
Multiple exclude patterns
tar -czf archive.tar.gz --exclude-vcs files/
Exclude version control files
tar -czf backup.tar.gz --exclude-caches files/
Exclude cache directories

üîç Verification & Testing

tar -tzf archive.tar.gz
Test gzip archive integrity
tar -tjf archive.tar.bz2
Test bzip2 archive integrity
tar -tf archive.tar | head -10
Show first 10 files in archive
tar -tf archive.tar | wc -l
Count files in archive
tar -tvf archive.tar | grep "pattern"
Search for files in archive

‚öôÔ∏è Advanced Options

tar -czf archive.tar.gz --strip-components=1 files/
Strip directory levels
tar -czf archive.tar.gz --transform 's/^/prefix-/' files/
Transform file names
tar -czf archive.tar.gz --owner=root --group=root files/
Set owner and group
tar -czf archive.tar.gz --numeric-owner files/
Use numeric IDs for owner/group
tar -czf archive.tar.gz --same-permissions files/
Preserve file permissions

üìä Archive Information

tar -tvf archive.tar | awk '{print $3}' | paste -sd+ | bc
Calculate total archive size
tar -tzf archive.tar.gz | grep "\.txt$" | wc -l
Count specific file types
tar -tvf archive.tar | sort -k6,7
Sort files by modification time
tar -tvf archive.tar | awk '{print $6, $7, $8}'
Show only timestamps and names

üíæ Backup Scenarios

tar -czf backup-$(date +%Y%m%d).tar.gz /home/user/
Daily backup with date
tar -czf backup.tar.gz --listed-incremental=backup.snar /data/
Incremental backup
tar -czf backup.tar.gz --newer-mtime="2024-01-01" files/
Backup files newer than date
tar -czf backup.tar.gz --exclude-backups files/
Exclude backup files
tar -czf - files/ | ssh user@remote 'cat > backup.tar.gz'
Stream backup to remote server

üîß Pipe Operations

tar -cf - files/ | tar -xf - -C /destination/
Copy files preserving structure
tar -czf - files/ | split -b 1G - backup.tar.gz.part
Split large archive into parts
cat backup.tar.gz.part* | tar -xzf -
Extract from split archive
tar -czf - files/ | gpg -c > encrypted.tar.gz.gpg
Create encrypted archive
gpg -d encrypted.tar.gz.gpg | tar -xzf -
Extract encrypted archive
