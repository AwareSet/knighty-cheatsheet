🚀 AWK Syntax & Structure
Basic AWK Command Structure
awk 'pattern { action }' file
awk -f script.awk file
command | awk 'pattern { action }'

Components:
• pattern - Condition to match (optional)
• { action } - Commands to execute (optional)
• If no pattern: action applies to all lines
• If no action: matching lines are printed
Pattern Type Syntax Description
BEGIN BEGIN { action } Execute before processing any input
END END { action } Execute after processing all input
Expression /regex/ { action } Match lines containing regex pattern
Range /start/,/end/ { action } Match lines from start to end pattern
Condition $1 == "value" { action } Match lines where condition is true

🛠️ Practical Examples
Print Specific Columns
awk '{print $1, $3}' file.txt
Print first and third columns only
Sum Column Values
awk '{sum += $2} END {print sum}' file.txt
Calculate sum of second column
Count Lines
awk 'END {print NR}' file.txt
Count total number of lines
Filter by Value
awk '$3 > 50 {print}' file.txt
Print lines where 3rd field > 50
Change Field Separator
awk -F',' '{print $1}' file.csv
Use comma as field separator
String Replacement
awk '{gsub(/old/, "new"); print}' file.txt
Replace all "old" with "new"
Add Line Numbers
awk '{print NR, $0}' file.txt
Add line numbers to output
Remove Empty Lines
awk 'NF > 0' file.txt
Remove blank/empty lines
Find Duplicates
awk '++count[$0] == 2' file.txt
Print duplicate lines only once
Calculate Average
awk '{sum+=$1; count++} END {print sum/count}' file.txt
Calculate average of first column
Print Specific Lines
awk 'NR>=10 && NR<=20' file.txt
Print lines 10 through 20
Transpose Data
awk '{for(i=1;i<=NF;i++) a[i,NR]=$i} END {for(i=1;i<=NF;i++) {for(j=1;j<=NR;j++) printf "%s ", a[i,j]; print ""}}'
Convert rows to columns

💡 Tips & Best Practices
🎯 Use Patterns
Combine patterns with actions for powerful filtering: /ERROR/ {count++}
📊 Track State
Use variables to maintain state across records for calculations and counting
🔧 Custom Separators
Use -F option or set FS variable for custom field separators
💾 BEGIN/END
Use BEGIN for initialization and END for final calculations
