🛠️ Basic Operations
العمليات الأساسية
Format Convert
تحويل التنسيق
ffmpeg -i input.avi output.mp4
Convert AVI to MP4
تحويل AVI إلى MP4
Resize Video
تغيير الحجم
ffmpeg -i input.mp4 -vf scale=1280:720 output.mp4
Resize to 720p
تغيير الحجم إلى 720p
Extract Audio
استخراج الصوت
ffmpeg -i input.mp4 -vn -c:a copy audio.aac
Extract audio only
استخراج الصوت فقط
Trim Video
قص الفيديو
ffmpeg -i input.mp4 -ss 00:01:30 -t 00:00:30 output.mp4
Cut 30s from 1:30
قص 30 ثانية من 1:30
🎥 Video Processing
معالجة الفيديو
Quality Control
التحكم في الجودة
ffmpeg -i input.mp4 -crf 23 output.mp4
Compress with CRF 23
ضغط بـ CRF 23
Add Watermark
إضافة علامة مائية
ffmpeg -i video.mp4 -i logo.png -filter_complex "overlay=10:10" output.mp4
Add logo at 10,10
إضافة شعار في 10,10
Rotate Video
تدوير الفيديو
ffmpeg -i input.mp4 -vf "transpose=1" output.mp4
Rotate 90° clockwise
تدوير 90° مع عقارب الساعة
Create GIF
إنشاء GIF
ffmpeg -i input.mp4 -vf "fps=10,scale=320:-1" output.gif
Convert to GIF 10fps
تحويل إلى GIF بـ 10 إطار/ث
🎵 Audio Processing
معالجة الصوت
Audio Quality
جودة الصوت
ffmpeg -i input.mp3 -b:a 128k output.mp3
Set bitrate 128kbps
تعيين معدل البت 128 كيلو بت/ث
Volume Control
التحكم في الصوت
ffmpeg -i input.mp4 -af "volume=0.5" output.mp4
Reduce volume 50%
تقليل الصوت 50%
Mix Audio
خلط الصوت
ffmpeg -i video.mp4 -i audio.mp3 -c:v copy -map 0:v:0 -map 1:a:0 output.mp4
Replace audio track
استبدال المسار الصوتي
Remove Audio
إزالة الصوت
ffmpeg -i input.mp4 -an output.mp4
Strip all audio
إزالة جميع الصوت
⚡ Advanced
عمليات متقدمة
Concatenate
دمج الفيديوهات
ffmpeg -f concat -safe 0 -i list.txt -c copy output.mp4
Merge videos from list
دمج الفيديوهات من قائمة
Stream RTMP
البث المباشر
ffmpeg -re -i input.mp4 -c copy -f flv rtmp://server/live/key
Stream to RTMP server
البث إلى خادم RTMP
Extract Frames
استخراج الإطارات
ffmpeg -i input.mp4 -vf fps=1 frame_%04d.png
1 frame per second
إطار واحد كل ثانية
Speed Change
تغيير السرعة
ffmpeg -i input.mp4 -filter:v "setpts=0.5*PTS" -filter:a "atempo=2" output.mp4
Double speed
مضاعفة السرعة

💡 Tips & Best Practices
نصائح وأفضل الممارسات
🚀 Performance
Use -c copy to avoid re-encoding when possible.
استخدم -c copy لتجنب إعادة الترميز عند الإمكان.
📊 Quality
CRF 18-23 for high quality, 23-28 for standard.
CRF 18-23 للجودة العالية، 23-28 للجودة القياسية.
🔍 Info
Use ffprobe to get file information.
استخدم ffprobe للحصول على معلومات الملف.
⚡ GPU
Use -hwaccel for hardware acceleration.
استخدم -hwaccel لتسريع الأجهزة.
📏 Aspect
Use scale=1280:-1 to maintain ratio.
استخدم scale=1280:-1 للحفاظ على النسبة.
🔄 Batch
Use shell loops for multiple files.
استخدم حلقات shell للملفات المتعددة.
