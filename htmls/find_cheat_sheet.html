Find Cheat Sheet

File Search Commands & Filters

üîç Basic Search

find . -name "filename"
Find files by exact name
find . -name "*.txt"
Find files by pattern
find . -iname "*.TXT"
Case-insensitive search
find /path -name "file*"
Search in specific directory
find . -type f -name "*.log"
Find files only (not directories)

üìÅ File Types

find . -type f
Find regular files
find . -type d
Find directories
find . -type l
Find symbolic links
find . -type b
Find block devices
find . -type c
Find character devices

üìè Size Filters

find . -size +100M
Files larger than 100MB
find . -size -1G
Files smaller than 1GB
find . -size 50c
Files exactly 50 bytes
find . -empty
Find empty files and directories
find . -size +10k -size -100k
Files between 10KB and 100KB

üïí Time Filters

find . -mtime -7
Modified within last 7 days
find . -mtime +30
Modified more than 30 days ago
find . -atime -1
Accessed within last 24 hours
find . -ctime -3
Changed within last 3 days
find . -mmin -60
Modified within last 60 minutes
find . -newer file.txt
Files newer than file.txt

üîí Permissions

find . -perm 755
Files with exact permissions 755
find . -perm -755
Files with at least 755 permissions
find . -perm +755
Files with any of 755 permissions
find . -perm -u+w
Files writable by owner
find . -executable
Find executable files
find . -readable
Find readable files

üë§ Ownership

find . -user username
Files owned by specific user
find . -group groupname
Files owned by specific group
find . -uid 1000
Files with specific user ID
find . -gid 1000
Files with specific group ID
find . -nouser
Files with no owner
find . -nogroup
Files with no group

‚öôÔ∏è Actions

find . -name "*.tmp" -delete
Delete matching files
find . -name "*.log" -exec rm {} \;
Execute command on each file
find . -name "*.txt" -exec grep "pattern" {} \;
Search content in found files
find . -type f -exec chmod 644 {} \;
Change permissions of found files
find . -name "*.bak" -ok rm {} \;
Confirm before executing command

üîó Logical Operators

find . -name "*.txt" -o -name "*.log"
OR condition (either match)
find . -name "*.txt" -a -size +1M
AND condition (both must match)
find . ! -name "*.txt"
NOT condition (negation)
find . \( -name "*.txt" -o -name "*.log" \) -size +1M
Grouped conditions with parentheses

üö´ Exclusions

find . -name "*.txt" -not -path "*/backup/*"
Exclude specific paths
find . -name "*.log" -prune -o -name "*.txt" -print
Exclude directories from search
find . -path "*/.*" -prune -o -name "*.txt" -print
Exclude hidden directories
find . -name ".git" -prune -o -type f -print
Exclude .git directories

üéØ Advanced Examples

find . -type f -name "*.txt" -exec wc -l {} + | sort -n
Count lines in files and sort
find . -type f -mtime +7 -size +100M -exec ls -lh {} \;
Large files older than 7 days
find . -name "*.log" -exec basename {} \; | sort | uniq -c
Count files by basename
find . -type f -printf "%s %p\n" | sort -nr | head -10
Find 10 largest files
