Golang Cheat Sheet

Essential Go Commands & Development Workflow

🚀 Basic Commands

go run main.go
Run Go program directly
تشغيل برنامج Go مباشرة
go build main.go
Compile Go program to binary
ترجمة برنامج Go إلى ملف تنفيذي
go build -o myapp main.go
Build with custom output name
البناء مع اسم مخصص للملف الناتج
go install
Compile and install package
ترجمة وتثبيت الحزمة
go version
Show Go version
عرض إصدار Go
go env
Show Go environment variables
عرض متغيرات بيئة Go

📦 Module Management

go mod init myproject
Initialize new module
إنشاء وحدة جديدة
go mod tidy
Clean up dependencies
تنظيف التبعيات
go mod download
Download dependencies
تنزيل التبعيات
go get package@version
Add specific package version
إضافة إصدار محدد من الحزمة
go get -u ./...
Update all dependencies
تحديث جميع التبعيات
go list -m all
List all module dependencies
عرض جميع تبعيات الوحدة

🧪 Testing

go test
Run tests in current package
تشغيل الاختبارات في الحزمة الحالية
go test ./...
Run tests in all packages
تشغيل الاختبارات في جميع الحزم
go test -v
Run tests with verbose output
تشغيل الاختبارات مع تفاصيل المخرجات
go test -cover
Run tests with coverage
تشغيل الاختبارات مع تقرير التغطية
go test -bench=.
Run benchmark tests
تشغيل اختبارات الأداء
go test -run TestName
Run specific test
تشغيل اختبار محدد

🏗️ Cross Compilation

GOOS=linux go build
Build for Linux
البناء لنظام Linux
GOOS=windows GOARCH=amd64 go build
Build for Windows 64-bit
البناء لنظام Windows 64-bit
GOOS=darwin go build
Build for macOS
البناء لنظام macOS
go env GOOS GOARCH
Show current OS and architecture
عرض نظام التشغيل والمعمارية الحالية
go tool dist list
List all supported platforms
عرض جميع المنصات المدعومة

🔧 Development Tools

go fmt ./...
Format all Go files
تنسيق جميع ملفات Go
go vet ./...
Check for common errors
فحص الأخطاء الشائعة
golint ./...
Check code style (external tool)
فحص أسلوب الكود (أداة خارجية)
go doc package
Show package documentation
عرض توثيق الحزمة
godoc -http=:6060
Start local documentation server
تشغيل خادم التوثيق المحلي
go clean
Remove build artifacts
إزالة ملفات البناء

🐛 Debugging & Profiling

go build -race
Build with race detection
البناء مع كشف حالات السباق
go test -race
Run tests with race detection
تشغيل الاختبارات مع كشف حالات السباق
go build -gcflags="-N -l"
Build for debugging (disable optimizations)
البناء للتصحيح (تعطيل التحسينات)
dlv debug
Start Delve debugger
تشغيل مصحح Delve
go tool pprof cpu.prof
Analyze CPU profile
تحليل ملف أداء المعالج
go tool trace trace.out
View execution trace
عرض تتبع التنفيذ

📊 Performance & Build

go build -ldflags="-s -w"
Build smaller binary (strip symbols)
بناء ملف تنفيذي أصغر (إزالة الرموز)
go build -tags production
Build with build tags
البناء مع علامات البناء
go build -a
Force rebuild all packages
إعادة بناء جميع الحزم قسرياً
time go build
Measure build time
قياس وقت البناء
go list -json
Get package info as JSON
الحصول على معلومات الحزمة كـ JSON

🌐 Web Development

go get github.com/gin-gonic/gin
Install Gin web framework
تثبيت إطار عمل Gin للويب
go get github.com/gorilla/mux
Install Gorilla Mux router
تثبيت موجه Gorilla Mux
go get gorm.io/gorm
Install GORM ORM
تثبيت GORM ORM
go run -race server.go
Run server with race detection
تشغيل الخادم مع كشف حالات السباق
go build -o server && ./server
Build and run server
بناء وتشغيل الخادم

🔍 Information & Help

go help build
Get help for build command
الحصول على مساعدة لأمر البناء
go list ./...
List all packages in project
عرض جميع الحزم في المشروع
go mod why package
Explain why package is needed
شرح سبب الحاجة للحزمة
go mod graph
Show module dependency graph
عرض رسم تبعيات الوحدة
go work init
Initialize workspace (Go 1.18+)
تهيئة مساحة العمل (Go 1.18+)
