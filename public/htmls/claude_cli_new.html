🧠 Claude CLI Cheat Sheet

Complete reference for Anthropic's Claude command-line interface
📦 Installation & Setup
Install Claude CLI
npm install -g @anthropic-ai/claude-cli
Install Claude CLI globally via npm
Alternative Install
pip install claude-cli
Install via Python pip package manager
Setup API Key
claude auth login
Authenticate with your Anthropic API key
Set API Key Environment
export ANTHROPIC_API_KEY="your-key-here"
Set API key as environment variable
Verify Installation
claude --version
Check installed Claude CLI version
💬 Basic Commands
Start Chat
claude chat
Start an interactive chat session
One-shot Query
claude "What is machine learning?"
Send a single query and get response
Chat with File Input
claude chat --file input.txt
Start chat with file content as context
Set Model
claude --model claude-3-sonnet
Specify which Claude model to use
Help
claude --help
Show all available commands and options
📁 File Operations
Analyze File
claude analyze code.py
Analyze and explain code file
Process Multiple Files
claude *.js "Review this code"
Process multiple files with query
Output to File
claude "Generate docs" > output.md
Save Claude's response to file
Watch File
claude watch file.py
Monitor file changes and auto-analyze
⚙️ Configuration
View Config
claude config show
Display current configuration
Set Default Model
claude config set model claude-3-opus
Set default Claude model
Set Temperature
claude config set temperature 0.7
Set response creativity (0.0-1.0)
Reset Config
claude config reset
Reset to default configuration
💭 Chat Features
Save Conversation
claude chat --save session.json
Save chat session to file
Load Conversation
claude chat --load session.json
Resume saved chat session
Set System Prompt
claude chat --system "You are a helpful coding assistant"
Set system prompt for chat session
Conversation History
claude history
View recent conversation history
Clear History
claude history clear
Clear conversation history
🚀 Advanced Options
Max Tokens
claude --max-tokens 1000 "Long query here"
Limit response length (tokens)
Stream Response
claude --stream "Tell me a story"
Stream response in real-time
JSON Output
claude --json "Generate data structure"
Format response as JSON
Verbose Mode
claude --verbose "Query here"
Show detailed debugging information
Timeout Setting
claude --timeout 60 "Complex query"
Set request timeout in seconds
🤖 Model Selection
Claude 3 Opus
claude --model claude-3-opus
Most capable model, best for complex tasks
Claude 3 Sonnet
claude --model claude-3-sonnet
Balanced performance and speed
Claude 3 Haiku
claude --model claude-3-haiku
Fastest model for simple queries
List Models
claude models list
Show all available models
🔧 Scripting & Automation
Batch Processing
claude batch --input queries.txt
Process multiple queries from file
Template Processing
claude template process.tmpl
Process template with variables
Pipeline Input
cat data.txt | claude "Summarize this"
Use stdin as input to Claude
Exit Codes
claude "query" && echo "Success"
Use exit codes for script control
💡 Pro Tip: API Key Security
Store your API key in environment variables or config files, never in your scripts. Use claude auth login for secure setup.
⚡ Performance Tip
Use Claude 3 Haiku for simple queries and batch processing. Reserve Opus for complex reasoning tasks that require the highest capability.
🔄 Workflow Integration
Integrate Claude CLI into your CI/CD pipelines for code review, documentation generation, and automated testing explanations.
📊 Monitoring Usage
Use --verbose flag to monitor token usage and API calls. Set appropriate timeouts for production scripts.
