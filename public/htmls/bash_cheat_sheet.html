🐚 BASH

Ultimate Interactive Cheat Sheet for Shell Scripting & Command Line
🚀 Basic Commands
Print Text Copy
echo "Hello World"
Display text or variables.
List Files Copy
ls -l
List directory contents in long format.
Change Directory Copy
cd /path/to/dir
Navigate to a directory.
Current Directory Copy
pwd
Print working directory.
Make Directory Copy
mkdir mydir
Create a new directory.
Remove File Copy
rm myfile.txt
Delete a file.
Remove Directory Copy
rm -r mydir
Delete a directory and its contents.
📁 File Operations
Copy File Copy
cp source.txt dest.txt
Copy a file.
Move/Rename File Copy
mv oldname.txt newname.txt
Move or rename a file/directory.
View File Content Copy
cat myfile.txt
Display file content.
View Paged Content Copy
less largefile.log
View large files page by page.
Edit File Copy
nano myfile.txt
Edit file using nano editor.
Change Permissions Copy
chmod +x myscript.sh
Make a script executable.
Change Ownership Copy
chown user:group myfile.txt
Change file owner and group.
➡️ Redirection & Pipes
Output to File file.txt')">Copy
echo "text" > file.txt
Redirect output to file (overwrite).
Append to File > file.txt')">Copy
echo "text" >> file.txt
Append output to file.
Pipe Output Copy
ls -l | grep ".txt"
Pipe output of one command to another.
Error Redirection Copy
command 2> error.log
Redirect stderr to a file.
All Output to File Copy
command &> output.log
Redirect stdout and stderr to a file.
📊 Variables & Environment
Set Variable Copy
MY_VAR="value"
Set a local variable.
Access Variable Copy
echo $MY_VAR
Access variable value.
Export Variable Copy
export PATH=$PATH:/new/path
Make variable available to subshells.
Command Substitution Copy
echo "Today is $(date)"
Execute command and use its output.
User Home Copy
echo $HOME
User's home directory.
Last Exit Status Copy
echo $?
Exit status of last command.
💡 Control Flow & Scripting
If Statement Copy
if [ -f "file.txt" ]; then echo "File exists" fi
Conditional execution.
For Loop Copy
for i in {1..5}; do echo $i done
Loop through a range or list.
While Loop Copy
count=0 while [ $count -lt 3 ]; do echo $count count=$((count+1)) done
Loop while a condition is true.
Case Statement Copy
case $VAR in "hello") echo "Hi";; "bye") echo "Goodbye";; *) echo "Default";; esac
Multi-way branching.
🔧 Functions & Aliases
Define Function Copy
my_func() { echo "Hello from function" }
Define a reusable function.
Call Function Copy
my_func
Execute a defined function.
Create Alias Copy
alias ll="ls -la"
Create a shortcut for a command.
List Aliases Copy
alias
Show all defined aliases.
⚙️ Process Management
List Processes Copy
ps aux
Display running processes.
Kill Process Copy
kill PID
Terminate a process by PID.
Background Process Copy
command &
Run a command in the background.
Foreground Process Copy
fg
Bring a background process to foreground.
🌐 Networking
Ping Host Copy
ping google.com
Check network connectivity.
Download File Copy
wget URL
Download files from the web.
Transfer Data Copy
curl URL
Transfer data with URL syntax.
Show IP Address Copy
ip a
Display network interface information.
💻 System Info
Disk Usage Copy
df -h
Report file system disk space usage.
Memory Usage Copy
free -h
Display amount of free and used memory.
Uptime Copy
uptime
Show how long the system has been running.
Kernel Version Copy
uname -a
Print system information.
📝 Text Processing
Search Text Copy
grep "pattern" file.txt
Search for patterns in files.
Count Lines/Words/Chars Copy
wc -l file.txt
Word, line, character count.
Sort Lines Copy
sort file.txt
Sort lines of text files.
Unique Lines Copy
uniq file.txt
Report or omit duplicate lines.
🛠️ Practical Examples
Find & Delete Files
find . -name "*.tmp" -delete
Find and delete all .tmp files recursively.
Backup Directory
tar -czf backup_$(date +%Y%m%d).tar.gz /path/to/dir
Create a gzipped tar archive with current date.
Monitor Logs
tail -f /var/log/syslog | grep "error"
Monitor syslog for lines containing "error".
Count Logins
who | wc -l
Count number of logged-in users.
Disk Space Alert
df -h | grep "/dev/sda1" | awk '{print $5}' | sed 's/%//g' | awk '{if($1 > 90) print "Disk space critical!"}'
Check disk usage and alert if over 90%.
Generate Random Password
head /dev/urandom | tr -dc A-Za-z0-9_@#%^&*+-= | head -c 16 ; echo
Generate a 16-character random password.
💡 Tips & Best Practices
🎯 Use Tab Completion
Press Tab for auto-completion of commands, file paths, and options.
⬆️ History Search
Use Ctrl+R to search through your command history.
📜 Script Headers
Start scripts with #!/bin/bash and add comments for clarity.
🔒 Quote Variables
Always quote variables (e.g., "$VAR") to prevent word splitting and globbing.
