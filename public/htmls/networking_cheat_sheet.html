Networking Cheat Sheet

Network Commands by OSI/TCP-IP Layers

🔌 Physical & Data Link Layer (L1-L2)

ip link show
Show network interfaces and status
عرض واجهات الشبكة والحالة
ip link set eth0 up
Bring interface up
تشغيل الواجهة
ethtool eth0
Show ethernet interface details
عرض تفاصيل واجهة الإيثرنت
iwconfig
Configure wireless interface
تكوين واجهة لاسلكية
iwlist scan
Scan for wireless networks
مسح الشبكات اللاسلكية
arp -a
Show ARP table (MAC addresses)
عرض جدول ARP (عناوين MAC)

🌐 Network Layer (L3) - IP & Routing

ip addr show
Show IP addresses on interfaces
عرض عناوين IP على الواجهات
ip addr add 192.168.1.100/24 dev eth0
Add IP address to interface
إضافة عنوان IP للواجهة
ip route show
Show routing table
عرض جدول التوجيه
ip route add default via 192.168.1.1
Add default route
إضافة مسار افتراضي
ping google.com
Test ICMP connectivity
اختبار اتصال ICMP
traceroute google.com
Trace ICMP route to destination
تتبع مسار ICMP إلى الوجهة

🚢 Transport Layer (L4) - TCP/UDP Ports

netstat -tuln
Show listening TCP/UDP ports
عرض منافذ TCP/UDP المستمعة
ss -tuln
Modern alternative to netstat
بديل حديث لـ netstat
lsof -i :80
Show processes using port 80
عرض العمليات التي تستخدم المنفذ 80
nc -zv google.com 80
Test TCP port connectivity
اختبار اتصال منفذ TCP
nc -l 8080
Listen on TCP port 8080
الاستماع على منفذ TCP 8080
telnet google.com 80
Connect to TCP port manually
الاتصال بمنفذ TCP يدوياً

📋 Application Layer (L5-L7) - DNS, HTTP, Services

nslookup google.com
DNS lookup
البحث في DNS
dig google.com
Detailed DNS query
استعلام DNS مفصل
dig @8.8.8.8 google.com MX
Query specific DNS server for MX records
استعلام خادم DNS محدد لسجلات MX
curl -I google.com
Get HTTP headers
الحصول على رؤوس HTTP
wget google.com
Download webpage via HTTP
تنزيل صفحة ويب عبر HTTP
whois google.com
Domain registration information
معلومات تسجيل النطاق

🔍 Network Scanning & Discovery

nmap -sP 192.168.1.0/24
Ping scan network range
مسح الشبكة بالفحص
nmap -sS 192.168.1.1
SYN scan for open ports
مسح SYN للمنافذ المفتوحة
nmap -p 22,80,443 192.168.1.1
Scan specific ports
مسح منافذ محددة
nmap -A 192.168.1.1
Aggressive scan with OS detection
مسح عدواني مع كشف نظام التشغيل
masscan -p1-65535 192.168.1.0/24
Fast port scanner
ماسح منافذ سريع

📊 Network Monitoring & Analysis

tcpdump -i eth0 host google.com
Capture packets to/from host
التقاط الحزم من/إلى المضيف
wireshark
GUI packet analyzer
محلل الحزم بواجهة رسومية
iftop
Real-time network bandwidth usage
استخدام النطاق الترددي في الوقت الفعلي
nethogs
Network usage by process
استخدام الشبكة حسب العملية
vnstat
Network statistics history
تاريخ إحصائيات الشبكة
mtr google.com
Continuous traceroute with stats
تتبع مستمر للمسار مع الإحصائيات

🔒 Firewall & Security

iptables -L
List firewall rules
عرض قواعد الجدار الناري
ufw status
Check UFW firewall status
فحص حالة جدار UFW الناري
ufw allow 22
Allow port through firewall
السماح للمنفذ عبر الجدار الناري
fail2ban-client status
Check fail2ban status
فحص حالة fail2ban
iptables -A INPUT -p tcp --dport 80 -j ACCEPT
Allow HTTP traffic
السماح لحركة HTTP

📡 Wireless Networks (WiFi)

nmcli dev wifi
List available WiFi networks
عرض شبكات WiFi المتاحة
nmcli dev wifi connect "SSID" password "password"
Connect to WiFi network
الاتصال بشبكة WiFi
iw dev wlan0 scan
Scan with iw tool
المسح باستخدام أداة iw
wavemon
Monitor wireless signal strength
مراقبة قوة الإشارة اللاسلكية
iwconfig wlan0 essid "NetworkName"
Set wireless network name
تعيين اسم الشبكة اللاسلكية

🔄 Network Services & DHCP

dhclient eth0
Renew DHCP lease
تجديد عقد DHCP
systemctl restart network-manager
Restart network manager
إعادة تشغيل مدير الشبكة
systemctl status networking
Check networking service status
فحص حالة خدمة الشبكة
resolvectl flush-caches
Flush DNS cache
مسح ذاكرة DNS التخزينية
systemd-resolve --status
Show DNS configuration
عرض إعدادات DNS

🌍 Internet Connectivity & Performance

curl ifconfig.me
Get public IP address
الحصول على عنوان IP العام
speedtest-cli
Internet speed test
اختبار سرعة الإنترنت
iperf3 -s
Start bandwidth test server
تشغيل خادم اختبار النطاق الترددي
iperf3 -c server_ip
Test bandwidth to server
اختبار النطاق الترددي للخادم
curl -w "%{time_total}" google.com
Measure HTTP response time
قياس وقت استجابة HTTP

🛠️ Advanced Tools & Utilities

socat TCP-LISTEN:8080,fork TCP:google.com:80
Port forwarding with socat
إعادة توجيه المنفذ باستخدام socat
nload
Real-time network load monitor
مراقب حمولة الشبكة في الوقت الفعلي
bmon
Bandwidth monitor and rate estimator
مراقب النطاق الترددي ومقدر المعدل
bandwhich
Modern network utilization tool
أداة حديثة لاستخدام الشبكة
sar -n DEV 1
Network device statistics
إحصائيات أجهزة الشبكة

⚡ Performance & Troubleshooting

ping -c 4 -i 0.2 google.com
Fast ping test for latency
اختبار ping سريع للكمون
traceroute -n google.com
Traceroute without DNS resolution
تتبع المسار بدون تحليل DNS
mtr --report-cycles 10 google.com
Network diagnostic report
تقرير تشخيص الشبكة
nslookup -debug google.com
DNS troubleshooting with debug
استكشاف أخطاء DNS مع التصحيح
timeout 5 nc -zv google.com 80
Port connectivity test with timeout
اختبار اتصال المنفذ مع مهلة زمنية
