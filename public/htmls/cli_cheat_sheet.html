CLI Commands Cheat Sheet

50 Most Essential Terminal Commands

📁 File Operations

ls -la
List files with details and hidden files
عرض الملفات مع التفاصيل والملفات المخفية
cp file.txt backup.txt
Copy file to new location
نسخ ملف إلى موقع جديد
mv oldname.txt newname.txt
Move/rename file
نقل أو إعادة تسمية ملف
rm -rf directory/
Remove directory and contents recursively
حذف مجلد ومحتوياته بشكل تكراري
touch newfile.txt
Create empty file or update timestamp
إنشاء ملف فارغ أو تحديث الطابع الزمني
ln -s /path/to/file symlink
Create symbolic link
إنشاء رابط رمزي

🗂️ Directory Navigation

cd /path/to/directory
Change to specific directory
الانتقال إلى مجلد محدد
cd ..
Go up one directory level
الانتقال للمجلد الأعلى بمستوى واحد
cd ~
Go to home directory
الانتقال للمجلد الرئيسي
pwd
Print current working directory
طباعة مسار المجلد الحالي
mkdir -p path/to/nested/dir
Create nested directories
إنشاء مجلدات متداخلة
rmdir empty_directory
Remove empty directory
حذف مجلد فارغ

📝 Text Processing

cat file.txt
Display file contents
عرض محتويات الملف
head -n 10 file.txt
Show first 10 lines of file
عرض أول 10 أسطر من الملف
tail -f logfile.log
Follow file changes in real-time
متابعة تغييرات الملف في الوقت الفعلي
grep "pattern" file.txt
Search for pattern in file
البحث عن نمط في الملف
sed 's/old/new/g' file.txt
Replace text in file
استبدال النص في الملف
awk '{print $1}' file.txt
Print first column of file
طباعة العمود الأول من الملف

💻 System Information

ps aux
List all running processes
عرض جميع العمليات الجارية
top
Display running processes dynamically
عرض العمليات الجارية بشكل ديناميكي
df -h
Show disk space usage
عرض استخدام مساحة القرص
du -sh *
Show directory sizes
عرض أحجام المجلدات
free -h
Display memory usage
عرض استخدام الذاكرة
uname -a
Show system information
عرض معلومات النظام

⚙️ Process Management

kill -9 1234
Force kill process by PID
إنهاء العملية قسرياً بواسطة رقم المعالج
killall process_name
Kill all processes by name
إنهاء جميع العمليات بالاسم
nohup command &
Run command in background
تشغيل الأمر في الخلفية
jobs
List active jobs
عرض المهام النشطة
bg %1
Put job in background
وضع المهمة في الخلفية
fg %1
Bring job to foreground
إحضار المهمة للمقدمة

🌐 Network & Transfer

curl -O https://example.com/file.zip
Download file from URL
تنزيل ملف من رابط
wget https://example.com/file.zip
Download file with wget
تنزيل ملف باستخدام wget
scp file.txt user@host:/path/
Copy file over SSH
نسخ ملف عبر SSH
rsync -av source/ dest/
Sync directories efficiently
مزامنة المجلدات بكفاءة
ping google.com
Test network connectivity
اختبار الاتصال بالشبكة
netstat -tuln
Show network connections
عرض اتصالات الشبكة

📦 Archive & Compression

tar -czvf archive.tar.gz folder/
Create compressed tar archive
إنشاء أرشيف tar مضغوط
tar -xzvf archive.tar.gz
Extract compressed tar archive
استخراج أرشيف tar مضغوط
zip -r archive.zip folder/
Create zip archive
إنشاء أرشيف zip
unzip archive.zip
Extract zip archive
استخراج أرشيف zip
gzip file.txt
Compress file with gzip
ضغط ملف باستخدام gzip
gunzip file.txt.gz
Decompress gzip file
إلغاء ضغط ملف gzip

🔐 Permissions & Users

chmod 755 script.sh
Change file permissions
تغيير صلاحيات الملف
chown user:group file.txt
Change file ownership
تغيير ملكية الملف
sudo command
Run command as administrator
تشغيل الأمر كمدير
su - username
Switch to another user
التبديل إلى مستخدم آخر
whoami
Show current username
عرض اسم المستخدم الحالي
id
Show user and group IDs
عرض معرفات المستخدم والمجموعة

🔍 Search & Find

find . -name "*.txt"
Find files by name pattern
البحث عن الملفات بنمط الاسم
locate filename
Quick file search using database
بحث سريع عن الملفات باستخدام قاعدة البيانات
which command
Find location of command
العثور على موقع الأمر
whereis command
Find binary, source, and manual
العثور على الملف التنفيذي والمصدر والدليل
grep -r "pattern" .
Search recursively in all files
البحث التكراري في جميع الملفات

📊 System Monitoring

htop
Interactive process viewer
عارض العمليات التفاعلي
iostat
I/O statistics
إحصائيات الإدخال والإخراج
vmstat
Virtual memory statistics
إحصائيات الذاكرة الافتراضية
lsof
List open files
عرض الملفات المفتوحة
uptime
Show system uptime and load
عرض وقت تشغيل النظام والحمولة

📜 Command History

history
Show command history
عرض تاريخ الأوامر
!!
Repeat last command
إعادة الأمر الأخير
!123
Repeat command number 123
إعادة الأمر رقم 123
ctrl+r
Search command history interactively
البحث في تاريخ الأوامر تفاعلياً
alias ll='ls -la'
Create command alias
إنشاء اختصار للأمر

📋 Text Utilities

sort file.txt
Sort lines in file
ترتيب الأسطر في الملف
uniq file.txt
Remove duplicate lines
إزالة الأسطر المكررة
wc -l file.txt
Count lines in file
عد الأسطر في الملف
cut -d',' -f1 file.csv
Extract first column from CSV
استخراج العمود الأول من ملف CSV
tr 'a-z' 'A-Z' < file.txt
Convert lowercase to uppercase
تحويل الأحرف الصغيرة إلى كبيرة
