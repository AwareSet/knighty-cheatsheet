üöÄ Basic Usage
Fetch a URL
curl https://example.com
Display the content of a URL.
Save to a File
curl -o file.html https://example.com
Save the output to a file with a specific name.
Save with Original Name
curl -O https://example.com/file.zip
Save the file with its original name.
üåê HTTP Methods
GET Request
curl https://api.example.com/users
Perform a GET request (default).
POST Request
curl -X POST -d "param1=value1" https://api.example.com/users
Perform a POST request with data.
POST JSON Data
curl -X POST -H "Content-Type: application/json" -d '{"key":"value"}' https://api.example.com/users
Send JSON data in a POST request.
PUT Request
curl -X PUT -d "data" https://api.example.com/users/1
Perform a PUT request to update a resource.
DELETE Request
curl -X DELETE https://api.example.com/users/1
Perform a DELETE request to remove a resource.
üì§ Headers & Data
Show Headers
curl -I https://example.com
Show the response headers only.
Include Headers in Output
curl -i https://example.com
Include response headers in the output.
Custom Header
curl -H "Authorization: Bearer token" https://api.example.com
Send a custom header with the request.
User Agent
curl -A "My User Agent" https://example.com
Specify a custom User-Agent string.
üîí Authentication
Basic Authentication
curl -u username:password https://example.com
Send a request with basic authentication.
Proxy Authentication
curl -x proxy.example.com:8080 -U user:pass https://example.com
Send a request via a proxy with authentication.
