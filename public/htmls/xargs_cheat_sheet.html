xargs Cheat Sheet

Command Construction & Parallel Execution

🚀 Basic Usage

echo "file1 file2" | xargs ls
Convert input to command arguments
تحويل المدخلات إلى معاملات أوامر
find . -name "*.txt" | xargs rm
Remove files found by find
حذف الملفات الموجودة بواسطة find
ls | xargs -I {} echo "File: {}"
Replace placeholder with input
استبدال العنصر النائب بالمدخل
echo "a b c" | xargs -n1 echo
Process one argument at a time
معالجة معامل واحد في كل مرة
cat files.txt | xargs cat
Concatenate files listed in input
دمج الملفات المذكورة في المدخل

⚡ Parallel Processing

find . -name "*.jpg" | xargs -P 4 -I {} convert {} {}.png
Process 4 files in parallel
معالجة 4 ملفات بشكل متوازي
echo {1..10} | xargs -n1 -P5 sleep
Run 5 parallel processes
تشغيل 5 عمليات متوازية
ls *.zip | xargs -P 0 -I {} unzip {}
Use maximum parallel processes
استخدام أقصى عدد من العمليات المتوازية
seq 1 100 | xargs -n1 -P10 curl -s
Make 10 parallel HTTP requests
إجراء 10 طلبات HTTP متوازية
find . -name "*.log" | xargs -P 8 gzip
Compress files with 8 parallel jobs
ضغط الملفات مع 8 مهام متوازية

🔧 Argument Control

echo "a b c d" | xargs -n2 echo
Process 2 arguments per command
معالجة معاملين لكل أمر
ls | xargs -L1 echo "Processing:"
Process one line at a time
معالجة سطر واحد في كل مرة
find . -name "*.txt" | xargs -r rm
Don't run if input is empty
عدم التشغيل إذا كان المدخل فارغاً
echo "file with spaces.txt" | xargs -d'\n' ls
Use newline as delimiter
استخدام السطر الجديد كفاصل
printf "a\0b\0c" | xargs -0 echo
Use null character as delimiter
استخدام الحرف الفارغ كفاصل

📁 File Operations

find . -name "*.bak" | xargs -I {} mv {} backup/
Move backup files to directory
نقل ملفات النسخ الاحتياطية إلى مجلد
ls *.txt | xargs -I {} cp {} {}.backup
Create backup copies of files
إنشاء نسخ احتياطية من الملفات
find . -name "*.log" -print0 | xargs -0 rm
Remove files with spaces in names
حذف الملفات التي تحتوي على مسافات في الأسماء
echo "dir1 dir2 dir3" | xargs mkdir
Create multiple directories
إنشاء مجلدات متعددة
find . -name "*.jpg" | xargs -I {} basename {}
Get filenames without paths
الحصول على أسماء الملفات بدون مسارات

🔍 Text Processing

find . -name "*.txt" | xargs grep "pattern"
Search pattern in multiple files
البحث عن نمط في ملفات متعددة
ls *.txt | xargs wc -l
Count lines in multiple files
عد الأسطر في ملفات متعددة
echo "file1.txt file2.txt" | xargs cat
Concatenate multiple files
دمج ملفات متعددة
find . -name "*.md" | xargs -I {} sed -i 's/old/new/g' {}
Replace text in multiple files
استبدال النص في ملفات متعددة
ls *.log | xargs -I {} tail -n 10 {}
Show last 10 lines of each file
عرض آخر 10 أسطر من كل ملف

📊 System Commands

ps aux | grep firefox | awk '{print $2}' | xargs kill
Kill processes by name
إنهاء العمليات بالاسم
docker ps -q | xargs docker stop
Stop all running containers
إيقاف جميع الحاويات الجارية
find /tmp -name "temp*" -mtime +7 | xargs rm
Remove old temporary files
حذف الملفات المؤقتة القديمة
ls /var/log/*.log | xargs -I {} du -h {}
Check size of log files
فحص حجم ملفات السجل
echo "user1 user2 user3" | xargs -n1 finger
Get info about multiple users
الحصول على معلومات عن مستخدمين متعددين

🌐 Network & Downloads

cat urls.txt | xargs -n1 -P5 curl -O
Download URLs in parallel
تنزيل الروابط بشكل متوازي
echo "google.com github.com" | xargs -n1 ping -c1
Ping multiple hosts
فحص اتصال مضيفين متعددين
cat servers.txt | xargs -I {} ssh {} 'uptime'
Check uptime on multiple servers
فحص وقت التشغيل على خوادم متعددة
seq 1 254 | xargs -P50 -I {} ping -c1 192.168.1.{}
Scan network range for live hosts
مسح نطاق الشبكة للمضيفين النشطين
cat images.txt | xargs -n1 -P10 wget
Download images with 10 parallel connections
تنزيل الصور مع 10 اتصالات متوازية

🛠️ Advanced Patterns

find . -name "*.c" | xargs -I {} sh -c 'echo "=== {} ==="; cat {}'
Execute shell command with header
تنفيذ أمر shell مع عنوان
printf '%s\n' *.txt | xargs -I {} bash -c 'echo "Lines in {}: $(wc -l < {})"'
Count lines with custom output
عد الأسطر مع مخرجات مخصصة
find . -name "*.jpg" | xargs -I {} sh -c 'convert {} $(dirname {})/thumb_$(basename {})'
Create thumbnails with path manipulation
إنشاء صور مصغرة مع معالجة المسار
echo "file1 file2" | xargs -I {} -P2 sh -c 'echo "Processing {}"; sleep 2; echo "{} done"'
Parallel execution with status updates
التنفيذ المتوازي مع تحديثات الحالة
seq 1 10 | xargs -I {} -P5 sh -c 'curl -s "api.com/item/{}" > item_{}.json'
Parallel API calls with file output
استدعاءات API متوازية مع إخراج الملفات

⚠️ Safety & Error Handling

find . -name "*.tmp" | xargs -r rm
Only run if input exists (-r flag)
التشغيل فقط إذا كان المدخل موجوداً (علامة -r)
echo "file1 file2" | xargs -t ls
Print commands before executing (-t flag)
طباعة الأوامر قبل التنفيذ (علامة -t)
find . -name "*.txt" | xargs -p rm
Prompt before executing (-p flag)
المطالبة قبل التنفيذ (علامة -p)
find . -name "*.log" -print0 | xargs -0 rm
Handle filenames with spaces safely
التعامل مع أسماء الملفات بمسافات بأمان
cat files.txt | xargs -I {} test -f {} && echo "{} exists"
Check if files exist before processing
فحص وجود الملفات قبل المعالجة
